Docker Compose Architecture Demo 🚀

This project demonstrates a small containerized architecture using Docker Compose.

Architecture Overview

  • Nginx acts as a reverse proxy and load balancer.
  • Static files are served directly by Nginx.
  • API requests are forwarded to two backend instances.
  • Load balancing happens between api1 and api2.

How It Works

  • Client → Nginx (single entry point)
  • Nginx → Reverse proxy to backend upstream
  • Requests are distributed between two containers
